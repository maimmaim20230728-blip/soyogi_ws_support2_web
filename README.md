# そよぎ 支援サポート（障害者版）

障害福祉の従事者のための、**支援のお守り**アプリ。
「福祉サポーターズサポートアプリ・そよぎ」シリーズの障害者版です。

> 支援の、お守りに。

## これは何

見えない障害（発達・精神、知的が絡む複合など）の現場で、**困った場面の対応を数秒で引けて、普段から読むと支援が上手くなる**ことを目指した、支援者向けの道具です。オフラインで動くPWA。**個人情報は一切持ちません**（記録はブラウザ端末内のみ）。

- 🚨 **こまった** … 場面カード12（緊急対応・外出時・報告・出典つき）。約30秒で残せる簡易記録つき
- 📖 **しらべる** … 障害特性11のリファレンス＋「このアプリの根拠」ページ
- 🌱 **まなぶ** … 今日のヒント（日替わり）＋ケースクイズ
- 🗣️ **つたえる** … トーキングエイド（定型文・50音）／筆談

## 設計の原則

1. **決めつけない** … すべて「傾向・ヒント」。目の前の本人が最優先
2. **根拠が辿れる** … 英NICE NG10/NG11、ポジティブ行動支援(PBS)、TEACCH、Epilepsy Foundation、CRPD/WHO QualityRights など国内外の枠組みを土台に
3. **医療はやらない** … 行動面の対応が領分。医療的な緊急は 119 と主治医の指示へ即誘導

## 技術

- 素の HTML / CSS / JS・PWA（ビルド不要）
- `content.ja.js` … 全コンテンツ（場面カード・特性・ヒント・クイズ・定型文）
- `app.js` … ロジック / `sw.js` … オフライン用 Service Worker（更新時は `CACHE` の版を上げる）
- `_check.js` … コンテンツの機械QA（`node _check.js`）
- `_make_icons.js` / `_make_embed.js` … アイコン・OGP画像の生成（sharp 使用）

初版は日本語のみ。対応の原則は普遍性が高いため、多言語展開は翻訳＋各国の緊急番号などの差し替えで対応予定。

## 公開

- Web 配信：Vercel（`soyogi-shien-support-web.vercel.app` を想定）
- Farcaster Mini App：`.well-known/farcaster.json` の `accountAssociation` は、デプロイ後に Manifest ツールで署名して差し込む
- Android：別途 Capacitor 版で AAB を作成予定
